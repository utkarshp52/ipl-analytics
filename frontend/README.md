# IPL Analytics - Frontend

React-based frontend for IPL Analytics System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Technologies

- React 18.2
- React Router DOM 6.x
- Axios
- Chart.js
- CSS3

## Features

- Responsive design
- Interactive charts
- Dynamic routing
- Real-time data from API
